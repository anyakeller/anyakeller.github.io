Tile Puzzle

A 3 by 3 sliding tile puzzle game built using Java. Clickable tiles allow sliding to complete the picture. Two sorting algorithms are available to solve the puzzle move by move.

Compile files using javac and run "$java PuzzleWindow" to play and "$java Puzzle" to see the algorithm in action
